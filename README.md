<h3 align="center">Project Summary</h3>
<p align="center"> The Mental Health Appointment System is a web-based backend application designed to streamline the connection between mental health professionals and patients seeking care. By providing a centralized platform for scheduling, the system aims to reduce administrative barriers to mental health support, allowing doctors to manage their availability and patients to book sessions efficiently. The application utilizes a Spring Boot architecture to manage core entities, ensuring a secure and organized approach to handling sensitive appointment data and user credentials.</p>
